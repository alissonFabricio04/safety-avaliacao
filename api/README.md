# Informação sobre a API 

## Requisitos

- PHP >= 7.1
- Mysql 

### Enviar os parâmetros abaixo usando o método POST 

 - token
 - name
 - mail
 - phone
 - cep
 - address_number
 - state
 - city
 - address 

### Na pasta `(./api)` rodar o comando abaixa

 - php -S localhost:8000
